    Rewrite `scripts/verify-svgs.js` to serve as a strict build step using native Node.js assertions:
    - Update the target directory from `../assets` to `../assets/images` to match where `generate-assets.js` outputs files.
    - Use Node's native `assert` module (e.g. `assert(/.../.test(content), '...')`) to throw immediate errors instead of manually counting them.

    - Assert that all SVGs contain `width="100%"`.
    - Assert that SVGs contain Frieren theme colors based on their filename: files ending in `-light.svg` must contain `#eab308` or `#0ea5e9` or `#bae6fd`; files ending in `-dark.svg` must contain `#94a3b8` or `#14b8a6` or `#581c87`.
    - Assert the existence of at least one generated `grimoire-*` SVG file to satisfy VISL-02 (e.g., `assert(files.some(f => f.startsWith('grimoire-')), ...)`.
  </action>
