## Anti-Patterns to Avoid
- **Forced Theme Integration:** Do not try to inject words like "Grimoire", "Spells", "Runes", or "Potions" into the text. The visual SVGs handle the theme; the text must remain professional and human.
- **Removing SVGs:** Do not alter the `<picture>` tags or image paths while refactoring the text.

## Common Pitfalls

### Pitfall 1: Breaking the Layout
**What goes wrong:** Markdown formatting is broken or SVGs are accidentally deleted during text replacement.
**Why it happens:** Careless find-and-replace or copy-pasting over HTML blocks.
**How to avoid:** Specifically target only the paragraph text nodes between the `<div align="center">` SVG blocks.

### Pitfall 2: Omitting the Crypto Backend Experience
**What goes wrong:** The Crypto Go backend experience is left out because it wasn't in the cover letter.
**Why it happens:** Strictly copying the cover letter without referencing `NARR-02`.
**How to avoid:** Explicitly synthesize a sentence about the Crypto Go backend that matches the cover letter's grounded tone.
